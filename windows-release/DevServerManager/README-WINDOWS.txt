# DevServer Manager v2.1.1 - Windows Release

## 🚀 Quick Start

1. **Extract** this ZIP file to your desired location
2. **Run** `DevServerManager.exe` to start the application
3. **Add servers** using templates or custom configuration

## 🔒 Important Security Notice

**Antivirus False Positive Warning**: Some antivirus software may flag this executable as suspicious. This is a **known false positive** common with PyInstaller applications.

**Why this happens:**
- Unsigned executable (no code signing certificate yet)
- Packed binary format triggers heuristic detection
- New application without established reputation

**✅ This application is safe:**
- **Open Source**: All code is publicly available on GitHub
- **No Network Data**: Application doesn't send data to external servers
- **Local Only**: All operations are performed locally

**If your antivirus flags this file:**
1. **Report false positive** to your antivirus vendor
2. **Add exclusion** for this file/folder temporarily
3. **Use source installation** if you prefer: https://github.com/idpcks/DevServerManager
